# Mobile E2E Smoke Tests

Run these manually on a staging build:

1. Launch app and confirm tenant branding appears.
2. Register or log in.
3. Browse categories and products.
4. Add item to cart.
5. Update cart quantity, remove an item, and confirm totals refresh.
6. Complete cash checkout.
7. Complete MTN checkout with a real staging number and verify success, failure, cancel, and timeout states.
8. Open orders list and order detail.
9. Submit a support request and verify retry after a failed request.
10. Receive a push notification on a physical device.
11. Log out and confirm the push token is unregistered or no longer targeted.
