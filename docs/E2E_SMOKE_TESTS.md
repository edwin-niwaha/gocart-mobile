# Mobile E2E Smoke Tests

Run these manually on a staging build:

1. Launch app and confirm tenant branding appears.
2. Register or log in.
3. Browse categories and products.
4. Add item to cart.
5. Complete checkout.
6. Open orders list and order detail.
7. Submit a support request.
8. Receive a push notification on a physical device.
9. Log out and confirm the push token is unregistered.
